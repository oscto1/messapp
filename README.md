#  Messapp (React + Firebase)

A chat room application built with React and Firebase. This project was originally created as a portfolio piece and demonstrates real-time messaging, Google authentication, and cloud-hosted deployment.

---

## Demo

[Click to go to the app](https://messa-69e78.firebaseapp.com/)
---

## Tech Stack

- React
- Firebase (Firestore)
- Firebase Authentication (Google)
- Firebase Hosting
- JavaScript (ES6+)

---

## Features

- Login with Google
- Create chat rooms
- Chat UI with message list rendering
- Cloud database integration via Firebase
- Responsive frontend UI
- Deployed on Firebase Hosting
